# Intuition

The longest common prefix of multiple strings is the longest substring that appears at the beginning of all strings. We can find it by starting with the first string as a candidate prefix and progressively shortening it until it matches the beginning of all other strings.

## Approach

1. Use the first string as the initial prefix candidate.
2. Iterate through the remaining strings in the array.
3. For each string, repeatedly remove the last character from the prefix until the current string starts with that prefix (using `startsWith`).
4. If the prefix becomes empty at any point, return an empty string immediately since there's no common prefix.
5. After processing all strings, return the final prefix.

## Complexity

- Time complexity: \(O(S)\), where \(S\) is the sum of all characters in all strings. In the worst case, we might need to check each character of each string.
- Space complexity: \(O(1)\), excluding the input array, since we only use a single string variable to store the prefix.

## Edge Cases

- **Empty array**: Should return an empty string (handled by the initial check).
- **Single string**: Returns that string itself as the prefix.
- **No common prefix**: Arrays like `["dog", "racecar", "car"]` should return `""`.
- **All strings identical**: Arrays like `["abc", "abc", "abc"]` should return `"abc"`.
- **One empty string**: Arrays containing `""` should return `""` since no non-empty prefix can be common.
- **Strings with different lengths**: Should correctly find the prefix up to the shortest matching length (e.g., `["flower", "flow", "flight"]` returns `"fl"`).

## Code

```typescript
function longestCommonPrefix(strs: string[]): string {
  if (!strs.length) return ''
  let prefix = strs[0]
  for (let i = 1; i < strs.length; i++) {
    while (!strs[i].startsWith(prefix)) {
      prefix = prefix.slice(0, -1)
    }
    if (prefix === '') return ''
  }
  return prefix
}
```

```typescript
// Without startsWith
function longestCommonPrefix(strs: string[]): string {
  if (!strs.length) return "";
  let prefix = strs[0];
  for (let i = 1; i < strs.length; i++) {
    let j = 0;
    while (
      j < prefix.length &&
      j < strs[i].length &&
      prefix[j] === strs[i][j]
    ) {
      j++;
    }
    prefix = prefix.slice(0, j);
    if (prefix === "") return "";
  }
  return prefix;
}
```
